## Default Permission

Default permissions for tauri-plugin-delta-comic.

#### This default permission set includes the following:

- `allow-get-safe-area-insets`
- `allow-hide-status-bar`
- `allow-lock-screen-orientation`
- `allow-set-status-bar-color`
- `allow-show-status-bar`
- `allow-unlock-screen-orientation`
